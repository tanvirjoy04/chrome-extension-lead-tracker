## How to use this extension 
1. Open chrome
2. On top-right corner, click on 3 dots. A drop-down list of options appears.
3. hover to "Extensions". Click "Manage Extensions".
4. On top-right corner just beside "Developer mode", click on toggle button and turn on "Developer mode".
5. On top-left corner click on "Load unpacked" button. Select location directory of this extension on your computer.
  - If you don't know how to download = clone a repo from github.
    1. Google 'how to git clone a repository'
    2. Facing trouble to understand? [Follow This.](https://www.theodinproject.com/lessons/foundations-git-basics)
6. Go to chrome. 
- On right side of website link bar, you'll find extension icon. 

- Click on that icon and you'll find my extension in a Green dollar sign icon. 

- Click on it and use this very useful extension.
